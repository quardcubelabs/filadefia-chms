# TAG Logo Setup Instructions

## Quick Setup

Please save the Tanzania Assemblies of God (TAG) logo image you provided as:

```
public/tag-logo.png
```

### Steps:
1. Save the TAG logo image file in the `public` folder
2. Name it exactly: `tag-logo.png`
3. The logo will automatically appear in:
   - Sidebar (hover-expandable navigation)
   - Dashboard header
   - Login page
   - Signup page  
   - Landing page (homepage)

### Logo Specifications:
- **Format**: PNG (with transparent background recommended)
- **Size**: Any size (will be auto-scaled)
- **Location**: `/public/tag-logo.png`

### Current Implementation:
The logo has been integrated into all pages with:
- White background container
- Border styling for visibility
- Responsive sizing
- Proper aspect ratio preservation
